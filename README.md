# FreeRTOS-Based-Real-Time-Equipment-Health-Monitoring-and-Predictive-Alert-System-using-LPC2129
An ARM7 (LPC2129) board runs FreeRTOS to sense a machine's temperature, load, and ambient light in real time, then hands the data off to a Linux server that logs it, predicts threshold breaches before they happen, and raises email + live-dashboard alerts the moment a real fault occurs.
